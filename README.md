# 냥빌리지
> 안양대학교 학생들을 위한 오픈 채팅 앱

~~~
자신이 수강하고 있는 과목의 질의응답을 쉽고 간편하게 할 수 있는 "냥빌리지"

냥빌리지는 아래와 같은 정보, 기능을 제공합니다.

- 회원가입 절차 없이 학교 포털사이트의 아이디와 비밀번호를 입력하면 로그인이 가능합니다.
- 자신이 듣는 수강 과목의 채팅방이 자동으로 개설됩니다.
- 푸시 알림 기능을 통해 메시지를 받을 수 있습니다.
- 채팅방 별 알림 허용/해제 할 수 있습니다.
- 악용하는 사용자의 메시지를 길게 누르면 신고할 수 있습니다.
~~~

## 🌄 배경
안양대학교 학생들은 편리한 시간표 생성, 학업 관리, 학교생활 정보 공유 등 다양한 기능을 제공하는 온라인 서비스를 이용하고 있지만, 정보 공유와 질문이 다소 포괄적이어서 제한적인 면이 있다는 것을 인식하게 되었습니다. 이에 따라, 학생들이 자신이 수강하는 과목에 대한 질문과 응답을 자유롭게 나눌 수 있는 안양대학교 전용 오픈 채팅 앱 "냥빌리지"를 개발하게 되었습니다.
